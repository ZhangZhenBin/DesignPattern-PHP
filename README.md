# 内聚：块内联系。
# 耦合：块间联系。
总结面向对象实际上就两句话:一是松耦合(Coupling),二是高内聚(Cohesion)。
面向对象系统追求的目标就是尽可能地提高系统模块内部的内聚(Cohesion)、尽可能降低模 块间的耦合(Coupling)。
